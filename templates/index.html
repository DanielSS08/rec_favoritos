Recuperar Favoritos

Veja como recuperar seus favoritos perdidos (Windows):

  1. Copie C:\Users\%username%\AppData\Local\Google\Chrome\User Data\Default para o Explorador de Arquivos.
  2. Na barra de pesquisa, digite "Bookmarks"; você verá uma lista de arquivos chamados Bookmarks e/ou Bookmarks.bak. (Observação: Se houver mais de um usuário usando o mesmo Chrome, os favoritos de outros usuários também serão listados.)
  3. Selecione todos os arquivos com o mouse e arraste-os para a área abaixo.
  4. Faça o download de todos os arquivos HTML.
  5. Abra cada arquivo HTML com o Chrome e identifique o arquivo HTML que contém seus favoritos. (Observação: O arquivo maior é provavelmente o correto.)
  6. No seu navegador Chrome, clique no ícone de menu do Chrome e vá para Favoritos > Gerenciador de Favoritos.
  7. Clique no ícone do menu ao lado da barra de pesquisa e clique em "Importar Favoritos".
  8. Selecione o arquivo HTML que contém seus favoritos.
  9. Seus favoritos devem agora ser importados de volta para o Chrome.
